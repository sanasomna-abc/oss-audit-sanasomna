## Project Overview
This repository contains a structured audit of the Python programming language, focusing on its role as an essential open-source component in the Linux ecosystem.

## Scripts Included
1. `system_identity.sh`: Reports OS and Kernel info.
2. `pkg_inspector.sh`: Checks for Python installation and licensing.
3. `dir_auditor.sh`: Audits Python-related system directories.
4. `log_analyzer.sh`: Scans logs for Python/System errors.
5. `manifesto_gen.sh`: Interactive Open Source philosophy generator.
